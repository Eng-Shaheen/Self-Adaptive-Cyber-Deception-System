# Running the Self-Adaptive Cyber Deception System

This guide explains how to deploy and test the Self-Adaptive Cyber Deception System in a local environment using Docker.

The system is containerized for safe execution and does not require manual dependency installation.

---

## Prerequisites

- Linux environment (tested on Kali Linux)
- Docker
- Docker Compose
- Internet access for pulling base images
